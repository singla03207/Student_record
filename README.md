# Student_record

----Student Record Management System in C----------

A beginner-level C programming project that stores and manages student records using structures, arrays, and functions.

---------Features-----------

* Store student details
* Store marks of 5 subjects
* Calculate average marks
* Generate grades automatically
* Display complete student information

-----Concepts Used-------

* Structures
* Arrays
* Functions
* Loops
* Conditional Statements

-----Technologies------

* C Programming Language
* GCC Compiler / VS Code
